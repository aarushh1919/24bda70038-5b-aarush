Aim of the Project — Student Management System

This project aims to design and develop a web-based Student Management System that efficiently manages student records through an interactive and user-friendly interface. The system allows users to add, view, and manage student information digitally, reducing manual work and improving data accuracy. It focuses on providing a scalable and responsive platform deployed on Vercel with planned MongoDB integration for secure data storage and retrieval.
